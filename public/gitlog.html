Como obter os commits
Rode um destes comandos no terminal do seu projeto:
# Última versão
git log --oneline --no-merges
# Entre tags
git log v1.0.0..HEAD --oneline --no-merges
# Formato completo
git log --pretty=format:"%s" --no-merges
